## How to Contribute

There are many ways to contribute:

- **Code:** Fix bugs, improve performance, or add new components.
- **Docs:** Improve documentation, write examples, or clarify usage.
- **Feedback:** Try the library, report issues, or suggest new ideas.
- **Design:** Help improve the look and feel of the components.

## Getting Started

1. Fork the repo.
2. Clone your fork locally.
3. Run `npm install` to install dependencies.
4. Make your changes.
5. Open a pull request and describe what you did.
